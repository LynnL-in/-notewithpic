道，它的发射与应用将空间技术发展推向了一个新的高度。望远镜的 $2.4\text{m}$ 镜头拥有所有镜头中最光滑的表面，其指向系统能在 $644\text{km}$ 以外将视野聚集在一枚硬币上。望远镜的偏差在 $1993$ 年 $12$ 月的一次太空任务中得到了大范围的校正。哈勃太空望远镜指向系统模型如图 3-54(a)所示，经简化后的结构图如图 3-54(b)所示。

**图 3-54 哈勃太空望远镜指向系统**

设计目标是选择放大器增益 $K_a$ 和具有增益调节的测速反馈系数 $K_1$，使指向系统满足如下性能：
1) 在阶跃指令 $r(t)$ 作用下，系统输出的超调量小于或等于 $10\%$。
2) 在斜坡输入作用下，稳态误差较小。
3) 减小单位阶跃扰动的影响。

**解** 由图 3-54(b)知，系统开环传递函数为
$$G(s) = \frac{K_a}{s(s+K_1)} = \frac{K}{s(s/K_1+1)}$$
式中，$K=K_a/K_1$ 为开环增益。
系统在输入与扰动同时作用下的输出为
$$C(s) = \frac{G(s)}{1+G(s)}R(s) + \frac{G_1(s)}{1+G(s)}N(s)$$
误差为
$$E(s) = \frac{1}{1+G(s)}R(s) - \frac{G_1(s)}{1+G(s)}N(s)$$

1) 满足系统对阶跃输入的要求。令
$$G(s) = \frac{K_a}{s(s+K_1)} = \frac{\omega_n^2}{s(s+2\zeta\omega_n)}$$

可得
$$\omega_n = \sqrt{K_a}, \quad \zeta = \frac{K_1}{2\sqrt{K_a}}$$
因为
$$\sigma\% = 100e^{-\pi\zeta/\sqrt{1-\zeta^2}}\%$$
解得
$$\zeta = \frac{1}{\sqrt{1+\frac{\pi^2}{(\ln\sigma)^2}}}$$
代入 $\sigma=0.1$，求出 $\zeta=0.59$，取 $\zeta=0.6$。因而，在满足 $\sigma\% \le 10\%$ 的指标要求下，应选
$$K_1 = 2\zeta\sqrt{K_a} = 1.2\sqrt{K_a}$$

2) 满足斜坡输入作用下稳态误差的要求。令 $r(t)=Bt$，由表 3-5 知
$$e_{ssr}(\infty) = \frac{B}{K} = \frac{BK_1}{K_a}$$
其 $K_a$ 与 $K_1$ 选择应满足 $\sigma\% \le 10\%$ 要求，即应有 $K_1=1.2\sqrt{K_a}$，故有
$$e_{ssr}(\infty) = \frac{1.2B}{\sqrt{K_a}}$$
上式表明，$K_a$ 的选取应尽可能大。

3) 减小单位阶跃扰动的影响。因为扰动作用下的稳态误差
$$e_{ssn}(\infty) = \lim_{s\to 0} sE_n(s) = -\lim_{s\to 0} sC_n(s)$$
$$= -\lim_{s\to 0} s \frac{G_1(s)}{1+G(s)}N(s) = -\lim_{s\to 0} s \frac{1}{s^2+K_1s+K_a} \cdot \frac{1}{s} = -\frac{1}{K_a}$$
可见，增大 $K_a$ 可以同时减小 $e_{ssn}(\infty)$ 及 $e_{ssr}(\infty)$。

在实际系统中，$K_a$ 的选取必须受到限制，以使系统工作在线性区。当取 $K_a=100$ 时，有 $K_1=12$，所设计的系统如图 3-55(a)所示；系统对单位阶跃输入和单位阶跃扰动的响应如图 3-55(b)所示。可以看出，扰动的影响很小。此时
$$e_{ssn}(\infty) = -0.01, \quad e_{ssr}(\infty) = 0.12B$$
得到了一个很好的系统。

MATLAB 文本：
```matlab
Ka=100; K1=12;
G1=zpk([], [0 -K1], 1);
sys=feedback(Ka*G1, 1);      %输入端闭环传递函数描述
sysn=feedback(G1, Ka);       %扰动端闭环传递函数描述
```
