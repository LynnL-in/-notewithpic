输出稳态误差 $e_{ss}(\infty) \leqslant 0.1\text{rad}$，开环系统截止频率 $\omega_c'' \geqslant 4.4\text{rad/s}$，相角裕度 $\gamma'' \geqslant 45^\circ$，幅值裕度 $h'' \geqslant 10\text{dB}$，试设计串联超前网络。

**解** 设计时，首先调整开环增益。因为
$$e_{ss}(\infty) = \frac{1}{K} \leqslant 0.1$$
故取 $K=10\text{rad}^{-1}$，则待校正系统开环传递函数
$$G_0(s) = \frac{10}{s(s+1)}$$

上式代表最小相位系统，因此只需画出其对数幅频渐近特性，如图 6-20 中 $L'(\omega)$ 所示。由图得待校正系统的 $\omega_c' = 3.1\text{rad/s}$，算出待校正系统的相角裕度为
$$\gamma = 180^\circ - 90^\circ - \arctan \omega_c' = 17.9^\circ$$

而二阶系统的幅值裕度必为 $+\infty\text{dB}$。相角裕度小的原因，是因为待校正系统的对数幅频渐近特性中频区的斜率为 $-40\text{dB/dec}$。由于截止频率和相角裕度均低于指标要求，故采用串联超前校正是合适的。

**[图 6-19 控制系统结构图]**
该图显示了一个反馈控制系统，输入信号 $R(s)$ 进入一个求和点，输出误差信号，该信号经过一个传递函数为 $\frac{K}{s(s+1)}$ 的控制器，输出为 $C(s)$。反馈路径直接连接输出端回到求和点。

**[图 6-20 例 6-3 系统开环对数幅频渐近特性]**
该图展示了对数幅频特性曲线 $L(\omega)$。横轴为频率 $\omega$（从 0.1 到 1000），纵轴为幅值 $L(\omega)$（从 -80 到 60）。
- 曲线 $L'(\omega)$ 在 $\omega=1$ 处通过 20dB，斜率为 -20dB/dec，在 $\omega=1$ 后变为 -40dB/dec。
- 曲线 $L_c(\omega)$ 为校正后的曲线，在 $\omega=4.4$ 处通过 0dB。
- 图中标记了 $\omega=2.2$ 处为 -20dB/dec 的起点，$\omega=8.8$ 处为 -40dB/dec 的起点。
- 标注了 $20\lg a = 12$ 的增益提升。

下面计算超前网络参数。试选 $\omega_m = \omega_c'' = 4.4\text{rad/s}$，由图 6-20 查得 $L'(\omega_m) = -6\text{dB}$，于是算得 $a = 4, T = 0.114\text{s}$。因此，超前网络传递函数为
$$G_c(s) = \frac{1+0.456s}{1+0.114s}$$

为了补偿无源超前网络产生的增益衰减，放大器的增益需提高 4 倍，否则不能保证稳态误差要求。

超前网络参数确定后，已校正系统的开环传递函数为
$$aG_c(s)G_0(s) = \frac{10(1+0.456s)}{s(1+0.114s)(1+s)}$$

其对数幅频渐近特性如图 6-20 中 $L''(\omega)$ 所示。显然，已校正系统 $\omega_c'' = 4.4\text{rad/s}$，算得待校正系统的 $\gamma(\omega_c'') = 12.8^\circ$，而由式(6-21)算出的 $\varphi_m = 36.9^\circ$，故已校正系统的相角裕度
$$\gamma'' = \varphi_m + \gamma(\omega_c'') = 49.7^\circ > 45^\circ$$

已校正系统的幅值裕度仍为 $+\infty\text{dB}$，因为其对数幅频特性不可能有限值与 $-180^\circ$ 线相交。此时，全部性能指标均已满足。

例 6-3 亦可采用惯性的 PD 控制器作为校正装置，它兼有直流放大器的功能。感兴趣的读者，不妨一试。

例 6-3 表明：系统串联校正后，中频区斜变为 $-20\text{dB/dec}$，并占据 $6.6\text{rad/s}$ 的频带范围，从而系统相角裕度增大，动态过程超调量下降。因此，在实际运行的控制系统中，中频区斜率大多具有 $-20\text{dB/dec}$ 的斜率。由例可见，串联超前校正可使开环系统截止频率增大，从而闭环系统带宽也增大，使响应速度加快。

应当指出，在有些情况下采用串联超前校正是无效的，它受以下两个因素的限制：
1) 闭环带宽要求。若待校正系统不稳定，为了得到规定的相角裕度，需要超前网络提供很大的超前相角。这样，超前网络的 $a$ 值必须选得很大，从而造成已校正系统带宽过大，使得通过系统的高频噪声电平很高，很可能使系统失控。
2) 在截止频率附近相角迅速减小的待校正系统，一般不宜采用串联超前校正。因为随着截止频率的增大，待校正系统相角迅速减小，使已校正系统的相角裕度改善不大，很难得到足够的超前相角。在一般情况下，产生这种相角迅速减小的原因，是在待校正系统截止频率的附近，或有两个交接频率彼此靠近的惯性环节；或有两个交接频率彼此相等的惯性环节；或有一个振荡环节。

在上述情况下，系统可采用其他方法进行校正，例如，采用两级(或两级以上)的串联超前网络(若选用无源网络，中间需要串联隔离放大器)进行串联超前校正，或采用一个滞后网络进行串联滞后校正，也可以采用测速反馈校正。

### 3. 串联滞后校正

利用滞后网络或 PI 控制器进行串联校正的基本原理，是利用滞后网络或 PI 控制器的高频幅值衰减特性，使已校正系统截止频率下降，从而使系统获得足够的相角裕度。因此，滞后网络的最大滞后角应力求避免发生在系统截止频率附近。在系统响应速度要求不高而抑制噪声电平性能要求较高的情况下，可考虑采用串联滞后校正。此外，如果待校正系统已具备满意的动态性能，仅稳态性能不满足指标要求，也可以采用串联滞后校正以提高系统的稳态精度，同时保持其动态性能仍然满足性能指标要求。

如果所研究的系统为单位反馈最小相位系统，则应用频域法设计串联无源滞后网络的步骤如下：
1) 根据稳态误差要求，确定开环增益 $K$。
2) 利用已确定的开环增益，画出待校正系统的开环对数频率特性，确定待校正系统的截止频率 $\omega_c'$、相角裕度 $\gamma(\omega_c')$ 和幅值裕度 $h(\text{dB})$。
3) 选择不同的 $\omega_c''$，计算或查出不同的 $b$ 值，在开环伯德图上绘制 $\gamma(\omega_c'')$ 曲线。
4) 根据相角裕度 $\gamma''$ 要求，选择已校正系统的截止频率 $\omega_c''$。考虑到滞后网络在新的截止频率 $\omega_c''$ 处会产生一定的相角滞后 $\varphi_c(\omega_c'')$，因此下式成立：
$$\gamma'' = \gamma(\omega_c'') + \varphi_c(\omega_c'') \quad (6-40)$$
式中，$\gamma''$ 是指标要求值，$\varphi_c(\omega_c'')$ 在确定 $\omega_c''$ 前可取为 $-6^\circ$。于是，根据(6-40)的计算结果，在 $\gamma(\omega_c'')$ 曲线上可查出相应的 $\omega_c''$ 值。
5) 根据下述关系式确定滞后网络参数 $b$ 和 $T$：
$$20\lg b + L'(\omega_c'') = 0 \quad (6-41)$$
$$\frac{1}{bT} = 0.1\omega_c'' \quad (6-42)$$
式(6-41)成立的原因是，因为要保证已校正系统的截止频率为上一步所选的 $\omega_c''$ 值，就必须使滞后网络的衰减量 $20\lg b$ 在数值上等于待校正系统在截止频率 $\omega_c''$ 上的对数幅频值 $L'(\omega_c'')$。该值在待校正系统对数幅频曲线上可以查出，于是由式(6-41)可以算出 $b$。
根据式(6-42)，由已确定的 $b$ 即可算出滞后网络的 $T$。如果求得的 $T$ 过大，难以实现，则可将式(6-42)中的系数 0.1 适当加大，例如在 0.1~0.25 范围内选取，而 $\varphi_c(\omega_c'')$ 的估计值相应在 $-6^\circ \sim -14^\circ$ 范围内确定。
6) 验算已校正系统的相角裕度和幅值裕度。

例 6-4 设计控制系统如图 6-21 所示。若要求校正后的系统的静态速度误差系数等于 $30\text{s}^{-1}$，相角裕度不低于 $40^\circ$，幅值裕度不小于 $10\text{dB}$，截止频率不小于 $2.3\text{rad/s}$，试设计串联校正装置。

**解** 首先，确定开环增益 $K$。由于
$$K_v = \lim_{s \to 0} sG(s) = K = 30\text{s}^{-1}$$
故待校正系统开环传递函数应取
$$G_0(s) = \frac{30}{s(1+0.1s)(1+0.2s)}$$

**[图 6-21 控制系统结构图]**
该图显示了一个反馈控制系统，输入信号 $R(s)$ 进入一个求和点，输出误差信号，该信号经过一个传递函数为 $\frac{K}{s(0.1s+1)(0.2s+1)}$ 的控制器，输出为 $C(s)$。反馈路径直接连接输出端回到求和点。

然后，画出待校正系统的对数幅频渐近特性，如图 6-22 所示。由 $\omega_c' = 12\text{rad/s}$ 算出
$$\gamma = 90^\circ - \arctan(0.1\omega_c') - \arctan(0.2\omega_c') = -27.6^\circ$$
说明待校正系统不稳定，且截止频率远大于要求值。在这种情况下，采用串联超前校正是无效的。可以证明，当超前网络的 $a$ 值取到 100 时，系统的相角裕度仍不满 $30^\circ$，而截止频率却增至 $26\text{rad/s}$。考虑到本例对系统截止频率要求不大，故选用串联滞后校正可以满足需要的性能指标。

现在作如下计算：
$$\gamma(\omega_c'') = 90^\circ - \arctan(0.1\omega_c'') - \arctan(0.2\omega_c'')$$
并将 $\gamma(\omega_c'')$ 曲线绘在图 6-22 中。根据 $\gamma'' \geqslant 40^\circ$ 要求和 $\varphi_c(\omega_c'') = -6^\circ$ 估值，按式(6-40)得 $\gamma(\omega_c'') \geqslant 46^\circ$。于是，由 $\gamma(\omega_c'')$ 曲线查得 $\omega_c'' = 2.7\text{rad/s}$。由于指标要求 $\omega_c'' \geqslant 2.3\text{rad/s}$，故 $\omega_c''$ 可在 $2.3 \sim 2.7\text{rad/s}$ 范围内任取。考虑取 $\omega_c''$ 值较大时，已校正系统响应速度较快，且滞后网络时间常数 $T$ 值较小，便于实现，故选取 $\omega_c'' = 2.7\text{rad/s}$。然后，在图 6-22 上查出当 $\omega_c'' = 2.7\text{rad/s}$ 时，有 $L'(\omega_c'') = 21\text{dB}$，故由式(6-41)求出 $b=0.09$，再由式(6-42)算出 $T=41\text{s}$，则滞后网络的传递函数
$$G_c(s) = \frac{1+bTs}{1+Ts} = \frac{1+3.7s}{1+41s}$$
