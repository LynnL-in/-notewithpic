$$\Phi_e(s) = \frac{E(s)}{R(s)} = 1 - \Phi(s) = \frac{1 - G_r(s)G(s)}{1 + G_r(s)G(s)} \quad (6-54)$$

上式亦可由式(6-51)直接导出。

在部分补偿情况下，$G_r(s) \neq 1/G(s)$。设反馈系统的开环传递函数
$$G(s) = \frac{K_v}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1)}$$

相应的闭环传递函数
$$\Phi(s) = \frac{K_v}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v} \quad (6-55)$$

显然，这是I型系统，存在常值速度误差，且加速度误差为无穷大。

若取输入信号的一阶导数作为前馈补偿信号，即
$$G_r(s) = \lambda_1 s$$

其中，常系数$\lambda_1$表示前馈补偿信号的强度。此时，由式(6-52)得等效系统的闭环传递函数
$$\Phi(s) = \frac{K_v(1 + \lambda_1 s)}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v} \quad (6-56)$$

根据式(6-54)，得等效系统的误差传递函数
$$\Phi_e(s) = \frac{a_n s^n + a_{n-1} s^{n-1} + \dots + a_2 s^2 + (a_1 - \lambda_1 K_v)s}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v}$$

取
$$\lambda_1 = \frac{a_1}{K_v}$$

可得
$$\Phi_e(s) = \frac{s^2(a_n s^{n-2} + a_{n-1} s^{n-3} + \dots + a_2)}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v} \quad (6-57)$$

于是，等效开环传递函数
$$G_k(s) = \frac{1 - \Phi(s)}{\Phi(s)} = \frac{a_2 s + K_v}{s^2(a_n s^{n-2} + \dots + a_2)}$$
*(注：原文此处公式排版有误，根据上下文逻辑应为：)*
$$G_k(s) = \frac{a_2 s + K_v}{s^2(a_n s^{n-2} + \dots + a_2)} \quad (6-58)$$

上式表明，引入$G_r(s)=\lambda_1 s$的前馈补偿装置，并使$\lambda_1=a_1/K_v$，可以使复合控制系统等效为II型系统。此时，复合控制系统的速度误差为零，加速度误差为常值。利用式(6-57)，根据终值定理方法不难验证这一结论。

若取输入信号的一阶导数和二阶导数的线性组合作为前馈补偿信号，即
$$G_r(s) = \lambda_2 s^2 + \lambda_1 s$$

则等效系统的闭环传递函数
$$\Phi(s) = \frac{K_v(1 + \lambda_1 s + \lambda_2 s^2)}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v} \quad (6-59)$$

等效系统的误差传递函数
$$\Phi_e(s) = \frac{a_n s^n + a_{n-1} s^{n-1} + \dots + (a_2 - \lambda_2 K_v)s^2 + (a_1 - \lambda_1 K_v)s}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v}$$

取
$$\lambda_1 = \frac{a_1}{K_v}, \quad \lambda_2 = \frac{a_2}{K_v}$$

可得
$$\Phi_e(s) = \frac{s^3(a_n s^{n-3} + a_{n-1} s^{n-4} + \dots + a_3)}{s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v} \quad (6-60)$$

于是，等效开环传递函数
$$G_k(s) = \frac{a_3 s^2 + a_2 s + K_v}{s^3(a_n s^{n-3} + a_{n-1} s^{n-4} + \dots + a_3)} \quad (6-61)$$

由式(6-60)及式(6-61)可见，引入$G_r(s)=\lambda_2 s^2 + \lambda_1 s$，并使$\lambda_1=a_1/K_v, \lambda_2=a_2/K_v$，可以使复合控制系统等效为III型系统。这时，复合控制系统的速度误差和加速度误差均为零，极大地提高了系统复现输入信号的能力和精度。

有时，前馈补偿信号不是加在系统的输入端，而是加在系统前向通路上某个环节的输入端，以简化误差补偿条件，如图6-33所示。由图可知，复合控制系统的输出量
$$C(s) = \frac{[G_1(s) + G_r(s)]G_2(s)}{1 + G_1(s)G_2(s)}R(s)$$

于是，等效系统的闭环传递函数
$$\Phi(s) = \frac{[G_1(s) + G_r(s)]G_2(s)}{1 + G_1(s)G_2(s)}$$

等效系统的误差传递函数
$$\Phi_e(s) = \frac{1 - G_r(s)G_2(s)}{1 + G_1(s)G_2(s)} \quad (6-62)$$

由上式可见，当取
$$G_r(s) = \frac{1}{G_2(s)} \quad (6-63)$$

时，复合控制系统将实现误差补偿。基于同样的理由，完全实现补偿条件(6-63)是困难的。为了使$G_r(s)$在物理上能够实现，通常只进行部分补偿，将系统误差减小至允许范围内即可。

从控制系统稳定性的角度考察，比较式(6-55)，式(6-56)和式(6-59)可知，没有前馈控制时的反馈控制系统的特征方程，与有前馈控制时的复合控制系统的特征方程完全一致，表明系统的稳定性与前馈控制无关。因此，复合控制系统很好地解决了一般反馈控制系统在提高控制精度与确保系统稳定性之间存在的矛盾。

在控制工程实践中，输入信号的一阶导数和二阶导数，往往由测速发电机与无源网络的组合线路取得，如图6-34所示。由图可知，在假定无源网络的负载阻抗为无穷大且信号源内阻为零，即$\beta R_w \approx 0$的条件下，其中$R_w$为测速发电机的分压电位器阻值，$\beta$为调整系数，无源网络的传递函数为
$$G_c(s) = \frac{K(Ts + 1)}{Ts + 1}$$
*(注：原文此处公式排版有误，根据上下文逻辑应为：)*
$$G_c(s) = \frac{K(Ts + 1)}{Ts + 1} \text{ (此处原文公式疑似印刷错误，应为 } \frac{K(\tau s + 1)}{Ts + 1} \text{)}$$
其中
$$K_1 = \frac{R_2}{R_1 + R_2}, \quad \tau = R_1 C_1, \quad T = \frac{R_1 R_2 C_1}{R_1 + R_2}$$

当不考虑负载效应时，整个测速发电机和无源网络的组合线路的传递函数为
$$G_r(s) = K_1 \frac{\tau s + 1}{Ts + 1} \cdot \beta K_s = \frac{\lambda_2 s^2 + \lambda_1 s}{Ts + 1} \quad (6-64)$$

式中
$$\lambda_1 = K_1 K_s \beta, \quad \lambda_2 = K_1 K_s \beta \tau$$
$K_s$为测速发电机的比电压(输出斜率)。显然，$\beta$和$\tau$的数值，可以使$\lambda_1$和$\lambda_2$满足设计要求。

由式(6-64)可见，由于无源网络无法提供纯微分信号，因此在前馈装置传递函数的分母上，增加了一项寄生因式$(Ts+1)$。在这种情况下，等效系统的闭环传递函数(6-59)演变为
$$\Phi(s) = \frac{K_v[1 + (\lambda_1 + T)s + \lambda_2 s^2]}{(Ts + 1)[s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v]} \quad (6-65)$$

根据(6-54)，得等效系统的误差传递函数
$$\Phi_e(s) = \frac{a_n T s^{n+1} + (a_n + a_{n-1} T)s^n + \dots + (a_2 + a_1 T - \lambda_2 K_v)s^2 + (a_1 - \lambda_1 K_v)s}{(Ts + 1)[s(a_n s^{n-1} + a_{n-1} s^{n-2} + \dots + a_2 s + a_1) + K_v]} \quad (6-66)$$

若取
$$\lambda_1 = \frac{a_1}{K_v}, \quad \lambda_2 = \frac{a_2 + a_1 T}{K_v} \quad (6-66)$$
也即$K_1, \beta$和$\tau$的选择，应使下列关系成立：
$$\beta = \frac{a_1}{K_1 K_s K_v}, \quad K_1 = \frac{a_2 + a_1 T}{K_1 K_s \beta \tau} \quad (6-67)$$

则等效开环传递函数
$$G_k(s) = \frac{(a_2 + a_1 T)s^2 + (a_1 + K_v T)s + K_v}{s^3[a_n T s^{n-2} + (a_n + a_{n-1} T)s^{n-3} + \dots + (a_3 + a_2 T)]} \quad (6-68)$$

上式表明，若条件式(6-66)或(6-67)成立，等效系统成为III型系统，其速度误差及加速度误差均为零。
